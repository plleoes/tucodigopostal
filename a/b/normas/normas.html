Normas de uso

  • Tu Código Postal nace solo para su uso en el ámbito de España.
  • Tu Código Postal se compromete a no utilizar la base de datos con fines comerciales o publicitarios(spam).
  • Esta web nace con vocación de servicio público y colaborativo.
  • Toda actividad comercial, negocio, servicio profesional o servicio entre particulares podrán ser dados de alta en su respectivo código postal, previo registro como usuario de la página, sin ninguna restricción excepto las legales.
  • Si en un comercio o servicio realiza mas de una actividad podrá dar de alta todas las que desee, o si es un particular que ofrece mas de un servicio tambien las puede dar de alta.
  • Si eres una empresa con varias sucursales podrá dar de alta la actividad en varios codigos postales.
  • Toda conexión de los usuarios registrados se almacenará en este servidor. Advertimos que toda actividad constitutiva de delito que se dé de alta en esta web se informará a la autoridad competente (Si observa Vd que en un código postal hay alguna actividad constitutiva de delito, informe de ella aqui).
